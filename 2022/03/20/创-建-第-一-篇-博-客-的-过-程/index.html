创 建 第 一 篇 博 客 的 过 程

create blog

首先建立一个存放博客文件和网站文件的文件夹
mkdir blog

然后 下载了 nodejs 和 hexo 后

初始化hexo–在blog中自动创建出一个网页

hexo init

然后 cnpm install –save hexo-deployer-git 下载git插件在blog中

（※）配置文件 将deploy下的文件配置为

type: ‘git’
repo: git@github.com:Lsta11/Lsta11.github.io.git(密钥法)
branch: main

然后生成ssh key 详情见https://blog.csdn.net/greenovia/article/details/60576985

注意： 在hexo d前需要设置Git的user name 和 user email
$ git config –global user.name “github的id”
$ git config –global user.email “github的邮箱”

然后 hexo clean hexo g hexo d

完成
